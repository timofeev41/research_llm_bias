# Базовые NLP метрики

В `calc.py` вычисляются следующие метрики для каждой пары (референс, ответ):

- **BLEU** (n-gram precision, BLEU-1, BLEU-2, BLEU-3, BLEU-4)
- **ROUGE-L** (precision, recall, f1)
- **BERTScore** (precision, recall, f1)
- **Длина ответа**:
    - по словам (`word_count`)
    - по токенам (`token_count`)
- **Лексическое разнообразие** (`lexical_diversity`)  
  (отношение числа уникальных токенов к общему числу токенов)
- **Доля уникальных n-грамм**:
    - `distinct_1` (уникальные униграммы)
    - `distinct_2` (уникальные биграммы)
- **Отношение длины ответа к длине референса** (`length_ratio`)

Все эти метрики считаются для каждого ответа каждой модели.
